# 🤖 AI Чат Поддержки - Руководство

## ✨ Что реализовано

Создан инновационный чат с AI поддержкой с красивыми анимациями и современным дизайном. Система использует многоуровневую архитектуру с fallback механизмом.

## 🏗️ Архитектура

Чат работает по принципу каскадного fallback:

1. **OpenAI API** (gpt-4o-mini) - основной AI
2. **Groq API** (llama-3.3-70b) - резервный AI
3. **Умные предустановленные ответы** - всегда работает
4. **Общее меню помощи** - последний fallback

## 📁 Созданные файлы

### 1. `components/SupportChat.tsx`
Главный компонент чата с:
- 🎨 Красивыми градиентами и анимациями (framer-motion)
- 💬 Модальным окном чата
- 🤖 Интеграцией с AI ботом
- ⚡ Плавными переходами и эффектами
- 🌙 Поддержкой темной темы
- 📱 Адаптивным дизайном

### 2. `app/api/chat/route.ts`
API endpoint с умной системой fallback:
- Пытается использовать OpenAI API
- При неудаче переключается на Groq API
- Если API недоступны - использует предустановленные ответы
- Всегда возвращает полезный ответ

### 3. `.env.local`
API ключи:
```env
OPENAI_API_KEY=sk-proj-...
GROQ_API_KEY=gsk_...
```

## 🎯 Функционал

### Умные предустановленные ответы

Система распознает ключевые слова и дает точные ответы:

**"доставка"** → Информация о способах доставки
- Курьерская доставка (1-3 дня)
- Почта России (5-10 дней)
- Самовывоз (бесплатно)

**"возврат"** → Правила возврата товаров
- Срок: 14 дней
- Условия возврата
- Сроки возврата денег

**"оплата"** → Способы оплаты
- Банковские карты
- Наличные при получении
- Apple Pay / Google Pay
- Оплата по счету

**"размер"** → Помощь с подбором размера
- Таблица размеров
- Как измерить параметры
- Бесплатный обмен

**"заказ"** → Как оформить заказ
- Пошаговая инструкция
- Что нужно для заказа

**"скидка"** → Акции и промокоды
- Скидка на первый заказ
- Бесплатная доставка
- Сезонные распродажи

**"отследить"** → Отслеживание заказа
- Через email
- В личном кабинете
- По телефону

- Пульсирующая анимация для привлечения внимания
- Плавное появление/исчезновение

- Индикатор печатания

## 🚀 Как использовать

### Для пользователей
1. Нажмите на кнопку чата справа снизу
2. Напишите свой вопрос
3. Нажмите Enter или кнопку отправки
4. Получите ответ от AI ассистента

### Примеры вопросов
- "Как оформить доставку?"
- "Как вернуть товар?"
- "Какие способы оплаты?"
- "Помогите с размером"
- "Есть ли скидки?"
- "Как отследить заказ?"

## 🎨 Дизайн особенности

### Цветовая схема
- Основной градиент: purple → pink → blue
- Пользователь: blue → purple
- AI: purple → pink
- Акценты: зеленый для статуса "Онлайн"

### Иконки (lucide-react)
- MessageCircle - кнопка чата
- Sparkles - AI индикатор
- Bot - аватар AI
- User - аватар пользователя
- Send - отправка сообщения
- X - закрытие чата

Полная поддержка темной темы через `dark:` классы Tailwind:

- Текст: `dark:text-gray-200`

- Не передаются на клиент

## 🚀 Деплой на Vercel

1. Добавьте переменные окружения (опционально):
   ```
   OPENAI_API_KEY=sk-proj-...
   GROQ_API_KEY=gsk_...
   ```

2. Деплой:
   ```bash
   vercel --prod
   ```

**Важно:** Чат будет работать даже без API ключей благодаря умным fallback ответам!
