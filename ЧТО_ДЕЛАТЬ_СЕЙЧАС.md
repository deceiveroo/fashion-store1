## 🚀 Следующие шаги

### 1. Дождитесь деплоя на Vercel
Vercel автоматически задеплоит последний коммит (1ffbb95).
Проверьте статус на https://vercel.com/

### 2. После успешного деплоя

#### A. Проверьте авторизацию админа

Подключитесь к базе данных и выполните:

```sql
-- Проверьте, есть ли админ
SELECT id, email, role FROM users WHERE role = 'admin';

-- Если нет, сделайте себя админом:
UPDATE users SET role = 'admin' WHERE email = 'ваш_email@example.com';
```

#### B. Войдите в систему
1. Откройте ваш сайт
2. Перейдите на `/auth/signin`
3. Войдите с email админа
4. Откройте `/admin/dashboard`

#### C. Проверьте, что ошибок нет
Откройте консоль браузера (F12):
- ✅ Не должно быть ошибок 401
- ✅ Не должно быть ошибок 500
- ✅ Все данные должны загружаться

### 3. Проверьте чат поддержки

#### Пользовательский чат:
1. Откройте главную страницу
2. Нажмите на кнопку чата (фиолетовый круг справа внизу)
3. Отправьте сообщение "привет"
4. Должен прийти ответ (fallback, так как нет Cloudflare AI)

#### Админ панель чата:
1. Войдите как админ
2. Откройте `/admin/support-chats`
3. Должен показаться список чатов
4. Выберите чат и попробуйте отправить сообщение

### 4. Настройте Cloudflare AI (опционально)

Если хотите умные ответы AI:

1. Зарегистрируйтесь на https://dash.cloudflare.com/
2. Включите Workers AI (бесплатно 10,000 запросов/день)
3. Получите Account ID и API Token
4. Добавьте в Vercel Environment Variables:
   ```
   CLOUDFLARE_ACCOUNT_ID=ваш_account_id
   CLOUDFLARE_API_TOKEN=ваш_api_token
   ```
5. Redeploy на Vercel

## 🐛 Если что-то не работает

### Ошибка 401 на админ роутах
```sql
-- Проверьте роль
SELECT id, email, role FROM users WHERE email = 'ваш_email';

-- Обновите роль
UPDATE users SET role = 'admin' WHERE email = 'ваш_email';
```

Затем:
1. Выйдите из системы
2. Очистите cookies
3. Войдите заново

### AI не отвечает
Это нормально без Cloudflare AI!
- Система использует fallback режим
- Отвечает на ключевые слова: доставка, возврат, оплата, размер, заказ, скидка
- Для умных ответов настройте Cloudflare AI (см. выше)

### Чат не открывается
1. Проверьте консоль браузера (F12)
2. Проверьте Network tab
3. Убедитесь, что таблицы БД существуют:
```sql
SELECT table_name FROM information_schema.tables 
WHERE table_schema = 'public' 
AND table_name IN ('support_chat_sessions', 'support_chat_messages');
```

### Vercel деплой не проходит
1. Проверьте логи на Vercel
2. Убедитесь, что все переменные окружения установлены:
   - DATABASE_URL
   - DIRECT_URL
   - NEXTAUTH_SECRET
   - JWT_SECRET
