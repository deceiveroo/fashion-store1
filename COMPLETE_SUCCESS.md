- Добавлена строка в `globals.css`

📄 Подробности: `DARK_THEME_FIX.md`

## ✨ Ключевые изменения в Tailwind v4

### 1. CSS-first конфигурация
```css
@import "tailwindcss";

/* Вместо tailwind.config.js */
@theme {
  --color-primary: #6366f1;
}
```

### 2. Темная тема
```css
/* Явная настройка варианта dark: */
@custom-variant dark (&:where(.dark, .dark *));
```

### 3. Автоматическое сканирование
Не нужно указывать `content: [...]` - Tailwind сканирует автоматически.

### 4. Oxide-движок
Написан на Rust для максимальной скорости.
