# Обновление структуры базы данных

В проект были внесены изменения в структуру базы данных для поддержки множественных категорий для товаров. Для применения изменений выполните следующие шаги:

## Шаги для обновления

1. **Остановите все запущенные процессы**
   - Остановите сервер разработки Next.js (если запущен)
   - Убедитесь, что другие процессы не используют файл `dev.db`

2. **Запустите скрипт инициализации базы данных**
   ```bash
   node init-db.js
   ```

3. **Запустите приложение**
   ```bash
   npm run dev
   ```

4. **Запустите сиды для добавления начальных данных**
   Перейдите по адресу: `http://localhost:3000/api/seed`

## Что изменилось

- Удалено поле `category` из таблицы `product`
- Добавлена таблица `category` для хранения категорий
- Добавлена таблица `product_category` для связи товаров и категорий (многие-ко-многим)
- Добавлены стандартные категории: Новинки, Коллекции, Мужское, Женское
- Обновлены API маршруты для работы с новой структурой
- Обновлены компоненты для отображения множественных категорий

## Возможные проблемы

Если возникнут проблемы с выполнением скрипта:
- Убедитесь, что Node.js установлен
- Проверьте, что все зависимости установлены (`npm install`)
- Убедитесь, что файл `dev.db` не используется другими процессами

После обновления структуры базы данных вы сможете добавлять товары с несколькими категориями через админ-панель.
