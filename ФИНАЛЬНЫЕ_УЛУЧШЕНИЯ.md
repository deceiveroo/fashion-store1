## Новые компоненты

### AnimatedBackground.tsx ✅
**Функциональность**:
- Универсальный компонент для анимированных частиц
- Настраиваемое количество частиц
- Настраиваемый цвет (purple, pink, blue, green, indigo)
- Поддержка темной темы
- Оптимизированные анимации

```tsx
<AnimatedBackground particleCount={25} color="purple" />
```

## Технические детали

### Темная тема
```css
/* Фоны страниц */
dark:from-gray-900 dark:via-purple-900/20 dark:to-gray-900

/* Карточки */
dark:bg-gray-800/80 dark:border-gray-700/50

/* Текст */
dark:text-white
dark:text-gray-300

/* Инпуты */
dark:bg-gray-700/70 dark:border-gray-600
dark:text-white dark:placeholder-gray-400

/* Hover эффекты */
dark:hover:bg-purple-900/20
```

### Анимации Framer Motion
```tsx
// Плавающие частицы
<motion.div
  animate={{
    y: [0, -100, 0],
    opacity: [0, 0.5, 0],
    scale: [0, 1.5, 0]
  }}
  transition={{
    duration: Math.random() * 5 + 3,
    repeat: Infinity
  }}
/>

// 3D hover эффекты
<motion.div
  whileHover={{ y: -5, scale: 1.02 }}
  whileTap={{ scale: 0.98 }}
/>
```
