## ✅ Проверка базы данных

### Таблицы существуют
```sql
SELECT table_name FROM information_schema.tables 
WHERE table_schema = 'public' 
AND table_name IN ('support_chat_sessions', 'support_chat_messages');
```
- [ ] Обе таблицы найдены

### Данные сохраняются
```sql
SELECT COUNT(*) FROM support_chat_sessions;
SELECT COUNT(*) FROM support_chat_messages;
```
- [ ] Количество растёт при отправке сообщений

### Связи работают
```sql
SELECT 
    scs.session_id,
    COUNT(scm.id) as message_count
FROM support_chat_sessions scs
LEFT JOIN support_chat_messages scm ON scs.session_id = scm.session_id
GROUP BY scs.session_id;
```
- [ ] Количество сообщений совпадает
