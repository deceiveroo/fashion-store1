## Ключевые улучшения

### Темная тема
```css
/* Фоны */
dark:from-gray-900 dark:via-purple-900/20 dark:to-gray-900

/* Карточки */
dark:bg-gray-800/80 dark:border-gray-700/50

/* Текст */
dark:text-white
dark:text-gray-300

/* Градиенты */
dark:from-purple-400 dark:via-pink-400 dark:to-blue-400
```

### Анимации Framer Motion
```tsx
// Плавающие частицы
<motion.div
  animate={{
    y: [0, -100, 0],
    opacity: [0, 0.5, 0],
    scale: [0, 1.5, 0]
  }}
  transition={{
    duration: Math.random() * 5 + 3,
    repeat: Infinity
  }}
/>

// Hover эффекты
<motion.div
  whileHover={{ scale: 1.01 }}
  whileTap={{ scale: 0.98 }}
/>

// Accordion анимации
<AnimatePresence>
  <motion.div
    initial={{ height: 0, opacity: 0 }}
    animate={{ height: "auto", opacity: 1 }}
    exit={{ height: 0, opacity: 0 }}
  />
</AnimatePresence>
```
