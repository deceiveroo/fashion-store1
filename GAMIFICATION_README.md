## 🎉 ЧТО РЕАЛИЗОВАНО

Полноценная система геймификации для модного магазина ELEVATE!

### ✅ Основные возможности:

**1. Система уровней** 📊

- Титулы: Новичок → Икона стиля

- 4 уровня редкости (Common, Rare, Epic, Legendary)

- Красивые карточки с анимациями

**3. Виртуальная валюта** 💰
- Монеты за квесты и достижения
- Обмен на скидки (100 монет = 100₽)
- Отображение баланса

**4. Ежедневные квесты** 🎯
- 5 ежедневных заданий
- Награды: XP + монеты
- Автоматический сброс

**5. Красивый UI** 🎨
- Градиенты и анимации
- Framer Motion эффекты
- Темная тема
- Адаптивный дизайн

## 🔧 ИНТЕГРАЦИЯ

### Начислить XP после покупки:

```typescript
// В вашем API после успешной покупки
import { db } from '@/lib/db';
import { sql } from 'drizzle-orm';

// Начислить XP
await db.execute(sql`
  UPDATE user_levels
  SET xp = xp + 50,
      coins = coins + 10,
      updated_at = NOW()
  WHERE user_id = ${userId}
`);

// Проверить повышение уровня
await db.execute(sql`
  UPDATE user_levels
  SET 
    level = level + 1,
    xp = xp - xp_to_next_level,
    xp_to_next_level = calculate_xp_to_next_level(level + 1),
    title = get_title_by_level(level + 1)
  WHERE user_id = ${userId} AND xp >= xp_to_next_level
`);
```

### Разблокировать достижение:

```typescript
// Проверить условие достижения
const purchaseCount = await getPurchaseCount(userId);

if (purchaseCount === 1) {
  // Разблокировать "Первые шаги"
  await db.execute(sql`
    INSERT INTO user_achievements (user_id, achievement_id)
    SELECT ${userId}, id FROM achievements WHERE code = 'first_purchase'
    ON CONFLICT DO NOTHING
  `);
  
  // Начислить награду
  await db.execute(sql`
    UPDATE user_levels
    SET xp = xp + 50, coins = coins + 10
    WHERE user_id = ${userId}
  `);
}
```

## 📊 ФОРМУЛЫ

### XP до следующего уровня:
```
XP = 100 * level^1.5
```

### Примеры:
- Уровень 1: 100 XP
- Уровень 5: 1,118 XP
- Уровень 10: 3,162 XP
- Уровень 20: 8,944 XP

```

20-29: Модный эксперт
```
