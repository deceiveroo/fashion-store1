## 🎯 Как работает новый чат:

### Polling вместо SSE:
- Каждые 2 секунды проверяет новые сообщения
- Более надежно чем SSE
- Работает везде (Vercel, локально, любой хостинг)
- Не блокирует соединение

### Умные ответы:
Чат распознает ключевые слова и дает готовые ответы:
- **"доставка"** → информация о доставке
- **"возврат"** → правила возврата
- **"оплата"** → способы оплаты
- **"размер"** → помощь с размером
- **"заказ"** → как оформить заказ
- **"скидка"** → акции и промокоды

### Cloudflare AI (опционально):
Если настроены переменные окружения:
```env
CLOUDFLARE_API_TOKEN=your_token
CLOUDFLARE_ACCOUNT_ID=your_account_id
```
Чат будет использовать Llama 3.3 70B для более умных ответов.

## 🔧 API Endpoints:

### POST /api/chat
Отправка сообщения:
```json
{
  "message": "Привет!",
  "sessionId": "session_123",
  "imageUrl": "data:image/png;base64,..." // опционально
}
```

### GET /api/chat?sessionId=xxx
Получение всех сообщений сессии

## 📊 Структура таблиц:

### support_chat_sessions
- id (UUID)
- session_id (уникальный)
- user_id (опционально)
- user_name, user_email
- status (active/resolved/closed)
- ai_disabled (если оператор взял управление)
- taken_over_by, taken_over_at
- resolved_by, resolved_at
- first_message
- message_count
- last_message_at
- rating (1-5 звезд)
- rated_at
- user_agent, ip_address
- created_at, updated_at

### support_chat_messages
- id (UUID)
- session_id
- user_id (опционально)
- message (текст)
- sender (user/ai/admin)
- ai_model (какая AI ответила)
- image_url (опционально)
- read_by_admin (прочитано админом)
- read_at
- created_at
