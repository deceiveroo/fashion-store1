# Улучшение страниц поддержки и компании

## Цель
Обновить все страницы поддержки (/support/*) и компании (/company/*) с:
- Полной поддержкой темной темы
- Более продвинутыми анимациями Framer Motion 12
- Интерактивными 3D эффектами
- Современными градиентами и эффектами свечения
- Улучшенной типографикой и читаемостью

## Страницы для обновления

### Support Pages
1. ✅ /support/contact - Контактная форма (частично обновлена)
2. /support/delivery - Информация о доставке
3. /support/returns - Возврат товаров
4. /support/sizes - Таблица размеров
5. /support/faq - Часто задаваемые вопросы

### Company Pages
1. /company/sustainability - Устойчивое развитие
2. /company/careers - Карьера
3. /company/press - Пресс-центр
4. /company/stores - Магазины

## Ключевые улучшения

### Темная тема
- Все фоны: `dark:from-gray-900 dark:via-gray-900 dark:to-gray-800`
- Карточки: `dark:bg-gray-800/80 dark:border-gray-700/50`
- Текст: `dark:text-white`, `dark:text-gray-300`
- Градиенты: добавить `dark:` варианты с более яркими цветами

### Анимации
- Parallax эффекты при скролле
- 3D tilt эффекты на карточках
- Плавные переходы между состояниями
- Анимированные градиенты
- Floating элементы

### Интерактивность
- Hover эффекты с трансформациями
- Ripple эффекты при клике
- Smooth scroll анимации
- Gesture поддержка для мобильных

## Технологии
- Framer Motion 12.23.24
- Tailwind CSS 4.0.7
- React 19.2.0
- Next.js 16.2.2
- Lucide React icons
