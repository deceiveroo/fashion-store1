## 🎨 Цветовая палитра темной темы:

### Фоны:
- `--background: #0a0a0f` - основной фон (очень темный синий)
- `--surface: #1a1a24` - поверхности (карточки, модалки)
- `--surface-hover: #252532` - hover состояние

### Текст:
- `--foreground: #f1f5f9` - основной текст (светлый)
- `--text-secondary: #94a3b8` - вторичный текст

### Акценты:
- `--primary: #818cf8` - основной цвет (светлый индиго)
- `--primary-hover: #a5b4fc` - hover состояние

### Границы:
- `--border: #2d2d3d` - границы элементов

## 📝 Как добавить темную тему в компоненты:

### Примеры классов:

```tsx
// Фон
className="bg-white dark:bg-gray-900"

// Текст
className="text-gray-900 dark:text-gray-100"
className="text-gray-600 dark:text-gray-400"

// Границы
className="border-gray-200 dark:border-gray-800"

// Карточки
className="bg-white dark:bg-gray-800 border border-gray-200 dark:border-gray-700"

// Кнопки
className="bg-purple-600 hover:bg-purple-700 dark:bg-purple-500 dark:hover:bg-purple-600"

// Инпуты
className="bg-white dark:bg-gray-800 border-gray-300 dark:border-gray-600 text-gray-900 dark:text-gray-100"

// Тени
className="shadow-lg dark:shadow-gray-900/50"
```
