# Настройка Telegram-бота для Fashion Store

## Проблема с подключением бота

Если вы столкнулись с ситуацией, когда:
- Написали в @ELEVATES_bot, но не получили ответа
- Кнопка "Позвать оператора" не передает сообщения в Telegram
- Не работает интеграция между веб-чатом и Telegram

Это означает, что вебхук Telegram-бота не был зарегистрирован.

## Шаги по настройке

### 1. Проверка токена бота

Вам нужен действующий токен бота. В проекте уже был определен токен:
```
7966709872:AAEDTXblN3sPVFEv-afFd4S0FjlLOA4b4DM
```

**ВАЖНО**: Этот токен был скомпрометирован (раскрыт в чате), поэтому его нужно отозвать и создать новый.

### 2. Отзыв старого токена и создание нового

1. Напишите в Telegram @BotFather
2. Отправьте команду `/revoke`
3. Выберите проблемный бот и подтвердите отзыв токена
4. Создайте нового бота командой `/newbot`
5. Получите новый токен

### 3. Обновление переменной окружения

Создайте или отредактируйте файл `.env.local` в корне проекта:
```
TELEGRAM_BOT_TOKEN=ваш_новый_токен_бота
```

### 4. Регистрация вебхука (для Windows PowerShell)

После развертывания приложения на публичном домене выполните команду в PowerShell:

```powershell
$token = "ваш_новый_токен_бота"
$webhookUrl = "https://ваш_домен.com/api/telegram/webhook"

$body = @{
    url = $webhookUrl
    allowed_updates = @("message", "callback_query")
} | ConvertTo-Json -Compress

Invoke-RestMethod -Uri "https://api.telegram.org/bot$token/setWebhook" -Method Post -Body $body -ContentType "application/json"
```

### 5. Для локальной разработки (используя ngrok)

1. Запустите приложение:
```bash
npm run dev
```

2. Запустите ngrok для проброса порта:
```bash
ngrok http 3000
```

3. Возьмите URL, который генерирует ngrok (например, `https://1234.ngrok.io`)

4. Зарегистрируйте вебхук с этим URL:
```powershell
$token = "ваш_новый_токен_бота"
$webhookUrl = "https://1234.ngrok.io/api/telegram/webhook"

$body = @{
    url = $webhookUrl
    allowed_updates = @("message", "callback_query")
} | ConvertTo-Json -Compress

Invoke-RestMethod -Uri "https://api.telegram.org/bot$token/setWebhook" -Method Post -Body $body -ContentType "application/json"
```

## Как работает интеграция

1. Когда пользователь нажимает "Позвать оператора" в веб-чате:
   - Сессия переводится в статус "ожидание оператора"
   - Уведомление отправляется в Telegram администраторам/операторам
   - В чате появляется сообщение: "Запрос передан специалисту..."

2. Оператор может:
   - Принять чат через Telegram-бота (нажав кнопку "Взять")
   - Отвечать пользователю через Telegram, и ответы будут отображаться в веб-чате

## Тестирование

После настройки вебхука:
1. Напишите боту любое сообщение - он должен ответить
2. Откройте веб-чат на сайте
3. Нажмите "Позвать оператора"
4. Проверьте, появилось ли уведомление в Telegram
