## 🐛 Известные проблемы

### ~~1. Переключение темной темы не работает~~
**Статус:** ✅ ИСПРАВЛЕНО

**Решение:** Добавлена директива `@custom-variant dark` в `globals.css`
- Tailwind CSS v4 требует явной настройки варианта `dark:`
- Добавлена строка: `@custom-variant dark (&:where(.dark, .dark *))`
- Темная тема теперь работает корректно

Подробности: `DARK_THEME_FIX.md`

### 1. Отсутствует колонка variant_id в order_items
**Статус:** Не критично

**Ошибка:**
```

```

**Решение:** Добавить колонку в таблицу order_items или удалить из запросов.

## 🌐 Деплой на Vercel

Проект полностью готов к развертыванию на Vercel!

### Созданные файлы для деплоя:
- ✅ `vercel.json` - конфигурация Vercel с оптимизациями
- ✅ `.vercelignore` - исключения для деплоя
- ✅ `.env.example` - пример переменных окружения
- ✅ `VERCEL_DEPLOYMENT.md` - подробное руководство (полная инструкция)
- ✅ `VERCEL_QUICK_START.md` - быстрый старт за 5 минут

### Быстрый деплой (5 минут):

1. **Импорт проекта:** https://vercel.com/new
2. **Выбрать репозиторий:** fashion-store
3. **Добавить переменные окружения:**
   ```bash
   DATABASE_URL=ваш_supabase_connection_pooling_url
   DATABASE_PASSWORD=ваш_supabase_password
   NEXTAUTH_URL=https://your-project.vercel.app
   NEXTAUTH_SECRET=сгенерируйте_openssl_rand_-base64_32
   NODE_TLS_REJECT_UNAUTHORIZED=0
   ```
4. **Нажать Deploy** и ждать ~3 минуты

### Оптимизации для Vercel:

✅ **Кеширование API** - настроено в vercel.json
✅ **Оптимизация изображений** - AVIF/WebP форматы
✅ **Code splitting** - автоматическое разделение бандлов
✅ **Compression** - gzip сжатие включено
✅ **Connection Pooling** - настроено для Supabase

### После деплоя:

1. Обновите `NEXTAUTH_URL` на реальный URL
2. Проверьте `/admin` и `/api/products`
3. Настройте кастомный домен (опционально)
4. Включите Vercel Analytics
5. Настройте мониторинг
