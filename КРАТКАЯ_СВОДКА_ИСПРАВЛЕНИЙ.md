## 🚀 Как запустить

### Быстрый старт:

1. **Перезапустите сервер:**
```bash
npm run dev
```

2. **Проверьте пользовательский чат:**
   - Откройте сайт
   - Нажмите на кнопку чата (фиолетовый круг)
   - Отправьте "привет"
   - Должен прийти ответ (AI или fallback)

3. **Проверьте админ панель:**
   - Войдите как админ
   - Откройте `/admin/support-chats`
   - Должен показаться список чатов
   - Выберите чат и попробуйте отправить сообщение

## ⚠️ Важно

### AI может не работать - это нормально!
Если не настроен Cloudflare AI, система использует встроенные ответы на ключевые слова:
- доставка
- возврат
- оплата
- размер
- заказ
- скидка

### Чтобы включить AI:
1. Зарегистрируйтесь на https://dash.cloudflare.com/
2. Включите Workers AI (бесплатно 10,000 запросов/день)
3. Получите Account ID и API Token
4. Добавьте в `.env`:
```env
CLOUDFLARE_ACCOUNT_ID=ваш_account_id
CLOUDFLARE_API_TOKEN=ваш_api_token
```
5. Перезапустите сервер

## 🔍 Проверка работы

### Логи сервера должны показывать:

**Если AI работает:**
```
[CHAT] Trying Cloudflare AI...
[CHAT] Cloudflare AI response: Привет! Чем могу помочь?
```

**Если AI не настроен (нормально):**
```
[CHAT] Cloudflare credentials not found
[CHAT] Using fallback response
```

**Если ошибка:**
```
[CHAT] Cloudflare AI error: ...
[CHAT] Using fallback response
```

## 🐛 Если что-то не работает

### Чаты не загружаются:
```sql
-- Проверьте роль админа
SELECT id, email, role FROM users WHERE email = 'ваш_email';

-- Если не admin, обновите:
UPDATE users SET role = 'admin' WHERE email = 'ваш_email';
```

### AI не отвечает (бесконечная загрузка):
1. Откройте консоль браузера (F12)
2. Проверьте вкладку Network
3. Найдите запрос `/api/chat`
4. Посмотрите ответ

### Сообщения не отправляются:
1. Проверьте консоль браузера (F12)
2. Проверьте логи сервера
3. Проверьте роль пользователя (должна быть 'admin')
