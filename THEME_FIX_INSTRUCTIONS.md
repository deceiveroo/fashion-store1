## 🔧 Технические детали

### Почему это работает?

В Tailwind CSS v4 изменился механизм работы с темной темой. Теперь нужно явно указать, что `dark:` классы должны применяться не только к элементу с классом `.dark`, но и ко всем его потомкам.

**Старый подход (Tailwind v3):**
```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

**Новый подход (Tailwind v4):**
```css
@import "tailwindcss";
@custom-variant dark (&:where(.dark, .dark *));
```

Селектор `&:where(.dark, .dark *)` означает:
- `&` - текущий элемент
- `.dark` - элемент с классом dark
- `.dark *` - ВСЕ потомки элемента с классом dark

### Как работает переключение?

1. При клике на кнопку ThemeToggle:
   - Меняется состояние `theme` в React
   - Добавляется/удаляется класс `dark` на `<html>`
   - Сохраняется выбор в `localStorage`

2. При загрузке страницы:
   - Скрипт в `<head>` читает `localStorage`
   - Если найдена тема `dark` - добавляет класс на `<html>`
   - Это происходит ДО рендера React (нет мигания)

3. Tailwind CSS видит класс `.dark` на `<html>` и применяет все `dark:` стили к потомкам

## 🐛 Если что-то не работает

### Проблема: Кнопка не переключает тему
**Решение:**
1. Очистите localStorage: `localStorage.clear()`
2. Перезагрузите страницу
3. Проверьте консоль на ошибки

### Проблема: Тема не сохраняется после перезагрузки
**Решение:**
1. Проверьте, что localStorage не заблокирован
2. Откройте DevTools → Application → Local Storage
3. Должен быть ключ `theme` со значением `light` или `dark`

### Проблема: Некоторые элементы не меняют цвет
**Решение:**
1. Убедитесь, что в `globals.css` есть строка `@custom-variant dark`
2. Перезапустите сервер разработки
3. Проверьте, что элемент имеет `dark:` классы

### Проблема: Мигание при загрузке страницы
**Решение:**
Это нормально при первой загрузке. Скрипт в `<head>` должен предотвращать это, но если мигание есть:
1. Проверьте, что скрипт в `app/layout.tsx` выполняется
2. Убедитесь, что он находится ПЕРЕД тегом `<body>`

## 📝 Дополнительные улучшения

Если хотите добавить системную тему (автоопределение):

```javascript
// В ThemeToggle.tsx
useEffect(() => {
  const savedTheme = localStorage.getItem('theme');
  if (savedTheme) {
    setTheme(savedTheme as 'light' | 'dark');
  } else {
    // Определяем системную тему
    const systemTheme = window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
    setTheme(systemTheme);
  }
}, []);
```
