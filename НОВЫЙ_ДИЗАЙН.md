# 🎨 Новый мега крутой дизайн

## ✨ Что нового

### Карточки товаров (ProductCard)

#### 3D Эффекты
- **3D Tilt** - карточка наклоняется при движении мыши
- **Parallax** - элементы двигаются на разных слоях
- **Transform 3D** - использует `transformStyle: "preserve-3d"`

#### Анимации
- **Gradient Border** - анимированная градиентная рамка при hover
- **Shimmer Effect** - блестящий эффект на изображении
- **Glow Effect** - свечение вокруг карточки
- **Floating Buttons** - кнопки плавно появляются снизу

#### Визуальные улучшения
- Rounded corners (rounded-3xl)
- Градиенты: purple → pink → blue
- Backdrop blur на кнопках
- Анимированные badges
- Улучшенные тени

// Scroll parallax
const { scrollYProgress } = useScroll();

## 🐛 Исправления

### Категории
**Было:** Отображались UUID
```
Категории: 5ccab42f-4b3c-4662-8bba-88558b398946
```

**Стало:** Отображаются названия
```
Категории: Одежда, Аксессуары
```

**Как исправлено:**
```typescript
// В app/products/[id]/page.tsx
const categoryNames = Array.from(
  new Set(
    productData
      .filter(item => item.categoryName)
      .map(item => item.categoryName as string)
  )
);
```
