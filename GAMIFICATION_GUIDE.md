# 🎮 Система геймификации - Руководство

## ✨ Что реализовано

Полноценная система геймификации для модного магазина с:
- Уровни и опыт (XP)
- Достижения (11 штук)
- Виртуальная валюта (монеты)
- Красивый UI с анимациями
- Редкость достижений (common, rare, epic, legendary)

## 📦 Структура

```
lib/db/
└── gamification-schema.ts     # Схема БД

app/api/gamification/
├── profile/route.ts            # API уровня пользователя
└── achievements/route.ts       # API достижений

components/gamification/
└── GamificationDashboard.tsx   # Основной компонент

app/gamification/
└── page.tsx                    # Страница геймификации

create-gamification-tables.sql  # SQL для создания таблиц
```

## 🚀 Установка

### 1. Создайте таблицы в БД:

```bash
psql -U your_user -d your_database -f create-gamification-tables.sql
```

Или выполните SQL вручную в вашем клиенте PostgreSQL.

### 2. Таблицы которые будут созданы:

- `user_levels` - уровни пользователей
- `achievements` - список достижений
- `user_achievements` - разблокированные достижения
- `daily_quests` - ежедневные квесты
- `user_quest_progress` - прогресс квестов
- `xp_history` - история получения опыта
- `leaderboard` - таблица лидеров

### 3. Запустите проект:

```bash
npm run dev
```

Откройте: `http://localhost:3000/gamification`

## 🎯 Система уровней

### Титулы по уровням:

- **Уровень 1-4**: Новичок
- **Уровень 5-9**: Любитель моды
- **Уровень 10-14**: Модник
- **Уровень 15-19**: Стиляга
- **Уровень 20-29**: Модный эксперт
- **Уровень 30-39**: Гуру стиля
- **Уровень 40-49**: Легенда моды
- **Уровень 50+**: Икона стиля

### Формула XP:

```
XP до следующего уровня = 100 * level^1.5
```

Примеры:
- Уровень 1 → 2: 100 XP
- Уровень 2 → 3: 283 XP
- Уровень 5 → 6: 1118 XP
- Уровень 10 → 11: 3162 XP

## 🔧 API Endpoints

### GET /api/gamification/profile
Получить уровень и статистику пользователя

```typescript
Response: {
  level: number;
  xp: number;
  xp_to_next_level: number;
  title: string;
  coins: number;
}
```

### GET /api/gamification/achievements
Получить все достижения с прогрессом

```typescript
Response: Achievement[] {
  id: string;
  name: string;
  description: string;
  icon: string;
  category: string;
  xp_reward: number;
  coins_reward: number;
  rarity: string;
  unlocked: boolean;
  unlocked_at?: string;
}
```

## 🎯 Интеграция

### Добавить XP пользователю:

```typescript
// В вашем API после покупки
await db.execute(sql`
  UPDATE user_levels
  SET xp = xp + 50,
      updated_at = NOW()
  WHERE user_id = ${userId}
`);

// Проверить повышение уровня
await db.execute(sql`
  UPDATE user_levels
  SET 
    level = level + 1,
    xp = xp - xp_to_next_level,
    xp_to_next_level = calculate_xp_to_next_level(level + 1),
    title = get_title_by_level(level + 1)
  WHERE user_id = ${userId} AND xp >= xp_to_next_level
`);
```

### Разблокировать достижение:

```typescript
await db.execute(sql`
  INSERT INTO user_achievements (user_id, achievement_id)
  VALUES (${userId}, ${achievementId})
  ON CONFLICT DO NOTHING
`);
```
