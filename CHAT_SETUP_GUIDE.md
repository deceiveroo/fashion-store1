## 📊 Что видит админ

В админ-панели вы увидите:

### Список чатов:
- Количество сообщений
- Статус (активный/решенный/архив)
- Первое сообщение (превью)
- Время последнего сообщения
- Кнопки "Решено" и "Архив"

### Просмотр сообщений:
- Все сообщения пользователя и AI
- Время отправки
- Какая AI модель ответила
- User Agent и IP (для аналитики)

## 🔍 Фильтры

- **Все** - все чаты
- **Активные** - требуют внимания
- **Решенные** - проблема решена
- **Архив** - старые чаты

## 💡 Как это работает

### Для пользователей:

1. Пользователь открывает чат
2. Генерируется уникальный `sessionId` (сохраняется в localStorage)
3. Все сообщения привязаны к этой сессии
4. Даже если закрыть и открыть чат - история сохраняется

### Для админов:

1. Все сообщения сохраняются в БД
2. Админ видит все чаты в реальном времени
3. Может отмечать чаты как решенные
4. Может добавлять заметки к чатам
