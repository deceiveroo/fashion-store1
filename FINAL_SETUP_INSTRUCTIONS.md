## 🔧 Если что-то не работает

### Ошибка "table does not exist"
→ Выполните SQL скрипт `add-chat-tables.sql` в Supabase

### Ошибка "The default export is not a React Component"
→ Перезапустите dev сервер (Ctrl+C, затем `npm run dev`)

### Чат не открывается
→ Проверьте консоль браузера (F12)
→ Убедитесь что нет ошибок JavaScript

### Админ-панель не показывает чаты
→ Убедитесь что вы вошли как админ
→ Проверьте что таблицы созданы в Supabase

## 📝 Структура БД

### Таблица `support_chat_sessions`:
- id - UUID сессии
- session_id - уникальный ID чата
- status - статус (active/resolved/archived)
- message_count - количество сообщений
- first_message - первое сообщение
- last_message_at - время последнего сообщения
- created_at - время создания

### Таблица `support_chat_messages`:
- id - UUID сообщения
- session_id - ID сессии
- message - текст сообщения
- sender - отправитель (user/ai)
- ai_model - какая AI модель ответила
- user_agent - браузер пользователя
- ip_address - IP адрес
- created_at - время отправки
