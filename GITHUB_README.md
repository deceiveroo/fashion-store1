# Fashion Store

Fashion Store - это современный интернет-магазин одежды, построенный на основе Next.js 14 с использованием App Router и различных методов оплаты.

## Возможности

- Поддержка множества способов оплаты:
  - Банковские карты (Visa, Mastercard, МИР) через Stripe
  - Система быстрых платежей (СБП)
  - Криптовалютные платежи (LTC, USDT TRC-20, TON, NOT)
  - Онлайн-банкинг (СБП, Qiwi, ЮMoney)
  - Наличные при получении
  - Рассрочка
- Административная панель с аналитикой
- Система управления товарами и заказами
- Интеграция с Telegram ботом
- Поддержка темной/светлой темы
- Адаптивный дизайн

## Технологии

- [Next.js](https://nextjs.org/) 14 с App Router
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Drizzle ORM](https://orm.drizzle.team/)
- [Supabase](https://supabase.io/) для хранения медиафайлов
- [Stripe](https://stripe.com/) для обработки платежей
- [NextAuth.js](https://next-auth.js.org/) для аутентификации
- [Shadcn UI](https://ui.shadcn.com/) компоненты

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/deceiveroo/fashion-store1.git
cd fashion-store
```

2. Установите зависимости:
```bash
npm install
```

3. Настройте переменные окружения:
```bash
cp .env.example .env.local
# Отредактируйте .env.local и добавьте свои значения
```

4. Запустите локальный сервер:
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере, чтобы увидеть результат.

## Переменные окружения

Для корректной работы приложения необходимо настроить следующие переменные окружения:

- `DATABASE_URL` - URL для подключения к базе данных
- `NEXTAUTH_URL` - URL приложения
- `NEXTAUTH_SECRET` - Секретный ключ для NextAuth
- `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET` - для аутентификации через Google
- `STRIPE_SECRET_KEY`, `NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY` - для работы с платежами Stripe
- `STRIPE_WEBHOOK_SECRET` - для обработки вебхуков Stripe

## Безопасность

Приложение было разработано с учетом лучших практик безопасности:

- Все чувствительные данные (ключи API, пароли от базы данных и т.д.) загружаются из переменных окружения
- Используется NextAuth.js для безопасной аутентификации
- Пароли пользователей хэшируются с использованием bcrypt

## Административная панель

Приложение включает полнофункциональную административную панель с возможностями:
- Управление товарами (добавление, редактирование, удаление)
- Просмотр и управление заказами
- Аналитика продаж
- Управление пользователями
- Настройки сайта

## Платежи

Приложение поддерживает множество способов оплаты:
- Stripe для оплаты банковскими картами
- Система быстрых платежей (СБП)
- Криптовалютные платежи (Bitcoin, Litecoin, Tether TRC-20, Toncoin)
- Интеграция с FK Wallet для удобства российских пользователей
