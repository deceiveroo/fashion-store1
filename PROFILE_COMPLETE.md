### 2. Реальные API Endpoints

#### `/api/profile/payments` - Способы Оплаты
- `GET` - Получить все сохраненные карты
- `POST` - Добавить новую карту
- `DELETE` - Удалить карту
- `PATCH` - Установить карту по умолчанию

#### `/api/profile/wishlist` - Избранное
- `GET` - Получить список избранного с данными товаров
- `POST` - Добавить товар в избранное
- `DELETE` - Удалить из избранного

#### `/api/profile/sessions` - Активные Сессии
- `GET` - Получить все активные сессии пользователя
- `DELETE` - Завершить сессию

#### `/api/profile/notifications` - Уведомления
- `GET` - Получить настройки уведомлений
- `POST` - Обновить настройки

#### `/api/profile/export` - Экспорт Данных (GDPR)
- `GET` - Скачать все данные пользователя в JSON
